## 1. ODE 損傷モデル（Tissue-Specific Damage-Repair）

### 1.1 基本方程式

組織特異的な損傷蓄積と修復を記述する非線形常微分方程式：

```
dD/dt = α · Load(t)^m − β · D(t) · exp(−τ · D(t))
```

| パラメータ | 意味 | デフォルト値 | 範囲 |
|-----------|------|-------------|------|
| α (alpha) | 損傷蓄積率 | 0.01 | [0.001, 0.1] |
| β (beta) | 修復率ベース | 0.5 | [0.1, 1.0] |
| τ (tau) | 修復飽和係数 | 2.0 | [0.5, 5.0] |
| m | 負荷-損傷の非線形性 | 2.0 | [1.0, 3.0] |
| D_crit | 臨界損傷閾値 | 0.8 | [0.6, 1.0] |

### 1.2 組織タイプ別パラメータ

| 組織 | α | β | τ | m | 根拠 |
|------|---|---|---|---|------|
| 筋肉 | 0.015 | 0.6 | 1.5 | 1.8 | 高い修復能力 |
| 腱 | 0.010 | 0.3 | 3.0 | 2.2 | 低血流による遅い修復 |
| 靭帯 | 0.008 | 0.25 | 3.5 | 2.5 | 最も修復が遅い |
| 骨 | 0.005 | 0.4 | 2.0 | 3.0 | 高い非線形性 |

### 1.3 数値解法

- **基本**: 前進オイラー法（日次タイムステップで十分安定）
- **高精度**: RK4（必要に応じて）
- **安定性条件**: dt * (α * Load_max^m + β) < 1

## 2. MRF（マルコフ確率場）リスク統合

### 2.1 概要

複数の損傷指標を統合し、全身のリスクマップを生成する。

### 2.2 エネルギー関数

```
E(x) = Σ_i ψ_i(x_i) + Σ_{(i,j)} φ_{ij}(x_i, x_j)
```

- **ψ_i**: 単項ポテンシャル（個別組織のリスク）
- **φ_{ij}**: ペアワイズポテンシャル（隣接組織間の相互作用）

### 2.3 推論

- Loopy Belief Propagation による近似推論
- 収束判定: メッセージ変化量 < 1e-6

## 3. 応力集中モデル（Stress Concentration）

### 3.1 局所応力

```
σ_local = K_t · σ_nominal
```

- **K_t**: 応力集中係数（解剖学的構造依存）
- **σ_nominal**: 公称応力（外部負荷から計算）

### 3.2 応力集中係数

| 部位 | K_t | 根拠 |
|------|-----|------|
| ハムストリング付着部 | 2.5 | 腱-骨接合部 |
| アキレス腱中間部 | 1.8 | 断面積変化 |
| 前十字靭帯 | 3.0 | 幾何学的集中 |
| 脛骨疲労骨折好発部 | 2.2 | 曲げ-捻り複合 |

## 4. サンプルエントロピー（SampEn）

### 4.1 目的

時系列データの複雑性（不規則性）を定量化し、過度の単調パターン（過負荷の兆候）を検出。

### 4.2 定義

```
SampEn(m, r, N) = -ln(A / B)
```

- **m**: 埋め込み次元（通常 m=2）
- **r**: 許容半径（通常 r=0.2*SD）
- **N**: データ長
- **A**: (m+1) 次元でマッチするペア数
- **B**: m 次元でマッチするペア数

### 4.3 解釈

| SampEn 値 | 解釈 | アクション |
|-----------|------|-----------|
| < 0.3 | 非常に規則的（懸念） | 負荷パターンの変更推奨 |
| 0.3 - 1.5 | 正常範囲 | 継続モニタリング |
| > 1.5 | 高い不規則性 | データ品質確認 |

## 5. EKF（拡張カルマンフィルタ）デカップリング検出

### 5.1 状態空間モデル

**状態方程式:**
```
x_k = F · x_{k-1} + B · u_k + w_k
```

**観測方程式:**
```
z_k = H · x_k + v_k
```

| 変数 | 意味 |
|------|------|
| x_k | 真の疲労状態 |
| u_k | 客観的負荷（GPS由来） |
| z_k | 主観的負荷（sRPE） |
| F | 状態遷移行列 |
| B | 入力行列 |
| H | 観測行列 |
| w_k | プロセスノイズ ~ N(0, Q) |
| v_k | 観測ノイズ ~ N(0, R) |

### 5.2 デカップリング判定

Innovation（残差）:
```
ν_k = z_k − H · x_k|k-1
```

Innovation の標準化:
```
ν_k_norm = ν_k / √(H · P_k|k-1 · H^T + R)
```

判定基準:
```
|ν_k_norm| > κ → デカップリング検出（κ = 0.85 デフォルト）
```

### 5.3 デカップリングの解釈

| Innovation 方向 | 意味 | 臨床的解釈 |
|----------------|------|-----------|
| ν < -κ·σ | sRPE が期待より低い | 過小申告の疑い（要注意） |
| ν > κ·σ | sRPE が期待より高い | 過大申告 or 体調不良の兆候 |

## 6. モデル統合パイプライン

```
Raw Data → ETL → ODE Engine → MRF Integration → Risk Score
                     ↓
              EKF Decoupling → Adjustment → Final Risk
                     ↓
              SampEn Analysis → Pattern Alert
```

### 6.1 リスクスコア統合

最終リスクスコアは加重統合:

```
Risk_final = w1 · D_ode + w2 · MRF_risk + w3 · Decoupling_flag + w4 · SampEn_alert
```

デフォルトウェイト: w1=0.4, w2=0.25, w3=0.25, w4=0.1

### 6.2 リスクカテゴリ

| スコア | カテゴリ | 色 | アクション |
|--------|---------|-----|-----------|
| 0.0 - 0.3 | 低リスク | GREEN | 通常練習可 |
| 0.3 - 0.6 | 中リスク | YELLOW | モニタリング強化 |
| 0.6 - 0.8 | 高リスク | ORANGE | 負荷制限推奨 |
| 0.8 - 1.0 | 極高リスク | RED | 練習中止推奨 |
