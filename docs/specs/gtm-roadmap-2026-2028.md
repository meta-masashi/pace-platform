# PACE v6.0 GTM Roadmap 2026-2028

## Executive Summary

PACE v6.0 の Go-To-Market 戦略は、3フェーズで「スポーツ医療の科学的意思決定プラットフォーム」としての
独占的ポジションを確立する。EBM-Bayesian エンジンを核に、7AM（朝7時レポート）の独占的価値提案から
データ民主化、そして因果推論による未来予測へと段階的に進化する。

---

## Phase 1: EBM-Bayesian Engine + 7AM Monopoly (2026 Q2-Q4)

### 目標
- **コールドスタート問題の解決**: 合成データ + バックテストで初期精度を証明
- **7AM レポートの独占的価値**: 朝7時に「今日の練習で誰がリスクか」を配信
- **投資家向け検証**: ROC-AUC ≥ 0.75 のバックテスト結果を提示

### コア機能
| 機能 | 説明 | KPI |
|------|------|-----|
| ODE 損傷モデル | 組織特異的な非線形損傷蓄積 | 損傷予測精度 ≥ 70% |
| EKF デカップリング検出 | 主観-客観乖離の統計的検出 | 偽陽性率 ≤ 15% |
| sRPE 過小申告アラート | 選手の申告バイアスを補正 | 検出率 ≥ 80% |
| 朝7時レポート | 日次リスクサマリー自動生成 | 配信率 100% |

### GTM 戦略
1. **ターゲット**: Jリーグ 5クラブ + 大学強豪 10チーム
2. **価格**: フリーミアム（基本無料、プレミアム機能は月額制）
3. **チャネル**: スポーツ医学会での発表 + 学術論文
4. **差別化**: 「エビデンスベース」を前面に、既存の勘と経験に対する科学的優位性

### マイルストーン
- 2026 Q2: バックテストパイプライン完成、投資家デモ
- 2026 Q3: パイロットクラブ 3チームで運用開始
- 2026 Q4: 初期結果の学会発表、シリーズA準備

---

## Phase 2: Data Democratization + Self-Evolution (2027)

### 目標
- **Edge AI 展開**: クラブ側のデバイスでリアルタイム推論
- **Bayesian オンライン学習**: 使うほど賢くなるモデル
- **データ連携エコシステム**: GPS/HR/ウェアラブルデータの統合

### コア機能
| 機能 | 説明 | KPI |
|------|------|-----|
| Edge 推論 | TFLite/ONNX でオンデバイス実行 | レイテンシ ≤ 100ms |
| オンライン学習 | Bayesian 逐次更新 | 精度向上 +5%/四半期 |
| データハブ | Catapult/Polar/Garmin 連携 | 連携デバイス ≥ 10種 |
| チーム間ベンチマーク | 匿名化統計での比較 | 参加チーム ≥ 30 |

### GTM 戦略
1. **ターゲット拡大**: プロリーグ全体 + 海外（K-League、A-League）
2. **価格**: SaaS モデル（選手あたり月額）
3. **パートナーシップ**: GPS デバイスメーカーとの API 連携
4. **差別化**: 「使うほど賢くなる」自己進化型プラットフォーム

### マイルストーン
- 2027 Q1: Edge AI プロトタイプ
- 2027 Q2: Bayesian オンライン学習の本番投入
- 2027 Q3: 海外パイロット開始
- 2027 Q4: シリーズB準備、ARR 目標達成

## Phase 3: DBN + Counterfactual Simulation (2028+)

### 目標
- **Dynamic Bayesian Network**: 時系列因果推論
- **反事実シミュレーション**: 「もし練習量を減らしていたら？」
- **規制対応**: 医療機器としての認証準備

### コア機能
| 機能 | 説明 | KPI |
|------|------|-----|
| DBN 因果モデル | 時系列因果グラフの自動学習 | 因果推論精度 ≥ 80% |
| What-If シミュレーション | 介入効果の事前予測 | シミュレーション速度 ≤ 1s |
| 医療グレード検証 | 臨床試験レベルの精度検証 | 感度 ≥ 90%, 特異度 ≥ 85% |
| レギュラトリー | SaMD (Software as Medical Device) 準備 | FDA/PMDA 事前相談完了 |

### GTM 戦略
1. **ターゲット**: 医療機関 + リハビリ施設 + 保険会社
2. **価格**: エンタープライズライセンス
3. **規制戦略**: SaMD クラスII 認証の取得
4. **差別化**: 「予測」から「介入効果の定量化」へ

### マイルストーン
- 2028 Q1: DBN エンジンの研究開発完了
- 2028 Q2: 反事実シミュレーション β版
- 2028 Q3: 医療機器認証プロセス開始
- 2028 Q4: 医療機関パイロット

## リスクと対策

| リスク | 影響 | 対策 |
|--------|------|------|
| データ不足 | モデル精度低下 | 合成データ + 転移学習 |
| 競合参入 | 市場シェア喪失 | 特許出願 + 学術的裏付け |
| 規制変更 | 市場アクセス制限 | PMDA との早期対話 |
| チーム採用抵抗 | 普及速度低下 | 無料トライアル + ROI 可視化 |
