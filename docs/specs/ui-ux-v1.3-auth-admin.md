## 4. ヘッダーロール切替UI（カテゴリB）

### 4.1 RoleSwitchToggle

スタッフ画面のヘッダーに表示。`staff_members` テーブルかつ `athletes` テーブルの両方にレコードがあるユーザーのみ表示。

#### 表示条件

- `staff_members` AND `athletes` に同一 `user_id` のレコードが存在する場合のみ表示
- 選手URLからログインした場合は非表示（選手→スタッフへの昇格禁止）
- スタッフURLからログインした場合のみ切替可能

#### ワイヤーフレーム

```
スタッフヘッダー（既存）:
┌──────────────────────────────────────────────────────────────┐
│  [PACE Logo]  チーム  選手  計画  シミュレータ  Analytics     │
│                                                              │
│                              [🔄 選手ビューに切替]  [👤 名前]│
│                              ← pill形状, bg-emerald-50,      │
│                                 text-emerald-700,            │
│                                 border-emerald-200           │
```
